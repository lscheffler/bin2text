Get_git_Path Function
Get installation status of git and paht to git binaries.
RetVal 
=
Get_git_Path
(@tc_git [,@tlIs64])
Parameter
tc_git
Path to git.exe.
Type Character
By reference  
Direction out
Includes git.exe
tlIs64
Is 64bit git installed.
Type Character
By reference  
Direction out
Optional  
Checks if 64bit is installed. 64bit needs different calling then 32bit, because VFP is 32bit.
Return value Boolean
Path found
Remarks
Checks HKLM\software\Microsoft\Windows\CurrentVersion\Uninstall\Git_is1\ for existence, returns the path out of InstallLocation value.
