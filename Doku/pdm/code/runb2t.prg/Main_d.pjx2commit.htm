Pjx2Commit Function
Create a commit to git.
RetVal 
=
Pjx2Commit
([tlAll])
Parameter
tlAll
Process whole path.
Type Boolean
Direction Input
Optional  
Return value Boolean
True if successfull.
Remarks
This calls FoxBin2PRG to process active project or all projects in the path. And runs a git commit.
