Convert_File Function
Runs FoxBin2Prg for a single file or class.
RetVal 
=
Convert_File
(tlText2Bin ,tlSingleClass [,tcFile [,tcClass]])
Parameter
tlText2Bin
Direction of operation.
Type Boolean
Direction Input

If true, create binaries from the text files corresponding tlSingleClass.

If false create text files.(Default)
tlSingleClass
Mode of file pickup, allows processing of single classes.
Type Boolean
Direction Input

If true, pick up a single class via AGETCLASS() or Input on tcFile and tcClass.

If false, GetFile() any source file (Default), or tcFile.(Default)

FoxBin2Prg will run in appropriate mode.
tcFile
File to process.
Type Numeric
Direction In
Optional  

Any file except project, what must be processed with Convert_Pjx.

Depending on tlText2Bin, input might be Binary or Binary and Text.

If tlSingleClass, in combination with tcClass.

Does not accept class in form library::class.
tcClass
Class name.
Type Numeric
Direction In
Optional  

Only with tcFile, if tlSingleClass is true.
Return value Boolean
Returns success of operation.
Remarks

Runs a pickup for a file and transforms it. Will always transform, neither the file is changed or not.

Picking a bin file for Text To Bin mode, tlText2Bin=.T., will select the txt file fitting. Same for selecting a class, it will process the classes text file if existing.

See also
Convert_Pjx
