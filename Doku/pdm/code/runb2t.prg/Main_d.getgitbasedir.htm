GetGitBaseDir Function
Returns the git root directory of a given directory.
RetVal 
=
GetGitBaseDir
([tcPath])
Parameter
tcPath
Path to investigate.
Type Character
Direction Input
Optional  
Default is FULLPATH(CURDIR())
Return value Character
Root directory of tcPath. Empty if tcPath is not under git control.
