Convert_Directory Function
Runs FoxBin2Prg for a directory.
RetVal 
=
Convert_Directory
(tlText2Bin [,tcDirectory])
Parameter
tlText2Bin
Direction of operation.
Type Boolean
Direction Input

If true, create binaries from the text files corresponding tcDirectory.

If false create text files.(Default)
tcDirectory
Directories.
Type Boolean
Direction Input
Optional  
Return value Boolean
Returns success of operation.
Remarks

Runs a pickup for a directory and transforms the files in it. Will always transform, neither the file is changed or not.

the files will be processed according to the setting of the config foxbin2prg.cfg file</p>
