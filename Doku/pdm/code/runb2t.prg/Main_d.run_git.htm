Run_git Function
Run git program
RetVal 
=
Run_git
(tcParameters [,tlShow [,tlIn_CMD]])
Parameter
tcParameters
Type Character
Direction Input
tlShow
Type Boolean
Direction Input
Optional  
tlIn_CMD
Type Boolean
Direction Input
Optional  

Some git commands like to be run in CMD, some dislike. Now we have a switch.

In general redirection of output (git status>xy.txt) needs CMD.
Return value Boolean
Success
Remarks
Runs API_AppRun to run git. Uses cmd to perform piping operations.
