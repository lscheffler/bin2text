Run_git Function
Run git program
RetVal 
=
Run_git
(tcParameters [,tlShow [,tlIn_Shell [,tlComplex]]])
Parameter
tcParameters
Type Character
Direction Input
tlShow
Type Boolean
Direction Input
Optional  
tlIn_Shell
Type Boolean
Direction Input
Optional  

Some git commands like to be run in CMD, some dislike. Now we have a switch.

In general redirection of output (git status>xy.txt) needs CMD.
tlComplex
Type Boolean
Direction Input
Optional  

Only if tlIn_Shell is false.

A rather complex command, like a pipe.
Return value Boolean
Success
Remarks
Run API_AppRun to run git. Uses cmd to perform piping operations.
