Convert_Array Function
Runs FoxBin2Prg for multiple files.
RetVal 
=
Convert_Array
(tlToBin [,tcMode ,@taFiles])
Parameter
tlToBin
Direction of operation.
Type Boolean
Direction Input

If true, create binaries from the text files corespondening taFiles.

If false create text files.(Default)
tcMode
Mode for pjx files.
Type Character
Direction Input
Optional  
Pjx ("*","*-","") parameters as used by FoxBin2Prg.
taFiles
Array with files
Type Array
By reference  
Direction Input

One column, full qualified filename of binary

Optional second column, if first colum is a project, projecthook.
Return value Boolean
True if successfull.
Remarks

Transfers a bunch of files

Processes binaries. If tlToBin the source files will be determined be the code.

Note that the files processed will be cached by the normal process of this programm. A operation that raises deletion like Delete obsolete files might delete text files without warning.
