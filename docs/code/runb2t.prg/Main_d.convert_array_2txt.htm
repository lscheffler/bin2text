Convert_Array_2Txt Function
Runs FoxBin2Prg for multiple files to text.
RetVal 
=
Convert_Array_2Txt
([tcMode ,@taFiles])
Parameter
tcMode
Mode for pjx files.
Type Character
Direction Input
Optional  
Pjx ("*","*-","") parameters as used by FoxBin2Prg.
taFiles
Array with files
Type Array
By reference  
Direction Input
<p>First column, full qualified filename of binary. or text.

Second column, if first column is a project, projecthook.

Third column, if first column is a table, a CHR(0) delimited list of index files, starting with CHR(0).
Return value Boolean
True if successfull.
Remarks

Transfers a bunch of files

Processes binaries.

Note that the files processed will be cached by the normal process of this programm. A operation that raises deletion like Delete obsolete files might delete text files without warning.
