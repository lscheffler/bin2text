Compare_VerNo Function
Compare Version numbers
RetVal

(tcVerNo1 ,tcVerNo2)
Parameter
tcVerNo1
Version number 1 in form nnnnn[.nnnnn[.nnnnn ... ]]]
Type Character
Direction Input
1 to 4 groups. Max 4 digits per group. Trailing groups will be ignored.

Version number 2 in form nnnn[.nnnn[.nnnn ... ]]]
Type Character
Direction Input
1 to 4 groups. Max 4 digits per group. Trailing groups will be ignored.
Return value Number
0

1

2
tcVerNo1 < tcVerNo2
