storage_get Method
Class: cusstorage
Description:Get settings information out of storage
Get settings from storage file.
RetVal 
=
Object.storage_get
(tcType ,toSettings ,tcStorage [,tcWhere])
Parameter
tcType
Type of the storage file to get settings value.
Type Character
Direction Input
toSettings
Settings object to write.
Type COLLECTION
Direction Input
tcStorage
Alias of the storage table.
Type Character
Direction Input
tcWhere
Expression for WHERE.
Type Character
Direction Input
Optional  
Used to filter records, in special the DBF for additional index per DBF (cType I).
Return value Boolean
At least one setting found.
Remarks

Every record in tcStorage marked with tcType will be read and stored into toSettings.

toSettings will be cleared before read.

For every item in toSettings a property of _SCREEN will be created. Form: _SCREEN.ADDPROPERTY(TRIM(cUniqueID),TRIM(cFile))
