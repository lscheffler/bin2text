storage_check Method
Class: cusstorage
Description:Check for valid storage settings, alter settings
Check program settings.
RetVal 
=
Object.storage_check
([tlInterActive [,tcHomePath]])
Parameter
tlInterActive
Interactive settings.
Type Boolean
Direction Input
Optional  
display the parent form to change the settings.
tcHomePath
Home path of he actual VFP IDE.
Type Character
Direction Input
Optional  
Return value Character
Alias of the storage table. .NULL. if error or tlInterActive.
Remarks

Opens the storage file and read settings. Test if settings are existing, not if valid.

If set tlInterActive the form will be opened to change and store settings.

The storage file will kept open if not tlInterActive, the alias will be returned if open.
