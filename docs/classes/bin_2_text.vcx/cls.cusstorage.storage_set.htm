storage_set Method
Class: cusstorage
Description:Set settings information to storage.
Puts settings into storage file.
Object.storage_set
(tcType ,toSettings ,tcStorage)
Parameter
tcType
Type of the storage file to assign settings value.
Type Character
Direction Input
toSettings
Settings object to read.
Type COLLECTION
Direction Input
tcStorage
Alias of the storage table.
Type Character
Direction Input
Remarks

Every item of the collection toSettings will create a record in tcStorage with key and value.

Note that lenght of key is limited to 14 signs, see Storage_Open.
