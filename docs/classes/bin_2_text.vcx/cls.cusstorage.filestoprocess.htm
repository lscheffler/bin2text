Checks if all files listed in tcStorage are used in taFiles.

Remarks

Parses taFiles for binaries that can be checked for changed. For now this will be scx and vcx files.

The actual state will be stored in a file Bin2Txt.DBF in tcPath path.

This file will hold information about any object of a vcx and information about scx.

The function will not handle any source file not found in the Array.

It is possible to delete splitted classes text file if the class is removed from an scx /vcx.

If tlCheckAll is true, text files for binary files removed might be deleted too.
