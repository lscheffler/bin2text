sync_file Method
Class: cusstorage
Description:Test change of checksum (crc32) for a given file (pair) against the storage table.
RetVal 
=
Object.sync_file
(tcType ,tcFile ,tlText2Bin ,tcStorage)
Parameter
tcType
Type of the storage file to sync the file.
Type Character
Direction Input
tcFile
File to check.
Type Character
Direction Input
tlText2Bin
Direction.
Type Boolean
Direction Input
If .T. files changed and new will be set as unmodified.
tcStorage
Alias to store actual values.
Type Character
Direction Input
Return value Boolean
tcFile is changed.
Remarks

The method calculates the checksum for the file and it's. memo file.

This will be compared with tcStorage to determine if the file is changed and need to be reprocessed.
