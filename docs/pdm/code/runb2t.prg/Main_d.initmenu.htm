InitMenu Function
Starts the IDE menu.
RetVal 
=
InitMenu
([tcHomePath] [,tlNoMenu])
Parameter
tcHomePath
Home path of he actual VFP IDE.
Type Character
Direction Input
Optional  
The default is not easy. If under git control, git root
tlNoMenu
Just read the IDE settings.
Type Boolean
Direction Input
Optional  
Return value Boolean
Remarks

Read the settings file in tcHomePath.

If the file not exists, create and initialize settings.

The VFP search path will be altered to keep the path to the application.

Places menu into _SYSMENU .
