Convert_Array_2Bin Function
Runs FoxBin2Prg for multiple files to binary.
RetVal 
=
Convert_Array_2Bin
([tcMode ,@taFiles])
Parameter
tcMode
Mode for pjx files.
Type Character
Direction Input
Optional  
Pjx ("*","*-","") parameters as used by FoxBin2Prg.
taFiles
Array with files
Type Array
By reference  
Direction Input

First column, full qualified filename of binary. or text. The column can declare a class in form File::Class as used by FoxBin2Prg for libraries.

Second column, if first column is a project, projecthook.

Third column, empty string.
Return value Boolean
True if successfull.
Remarks

Transfers a bunch of files

Processes binaries. The source files will be determined be the code.

Note that the files processed will be cached by the normal process of this programm. A operation that raises deletion like Delete obsolete files might delete text files without warning.
