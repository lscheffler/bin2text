Convert_Class_2Bin Function
Runs FoxBin2Prg for a single class to create binary class library.
Convert_Class_2Bin
(tlSingleClassLib ,tlTextFile)
Parameter
tlSingleClassLib
Mode of file output, allows processing to a single class library.
Type Boolean
Direction Input

If true, out of the file[.baseclass].class.tx2 a file[.baseclass].class.VCX class will be created.

If false, the class will be added to library file.VCX. Other classes will not be touched.
tlTextFile
Mode of file query, query for a text file or use AGETCLASS() .
Type Boolean
Direction Input

<p> Runs a pickup for a VCX based class and transforms it.

FoxBin2Prg will run in appropriate mode. This is



  • RedirectClassPerFileToMain: 1

  • RedirectClassType: 2 ; for tlSingleClassLib true

  • RedirectClassType: 1 ; for tlSingleClassLib false


</remarks>
