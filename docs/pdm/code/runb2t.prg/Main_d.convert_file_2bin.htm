Convert_File_2Bin Function
Runs FoxBin2Prg for a single file or class to create binary files.
RetVal 
=
Convert_File_2Bin
(tlSingleClass [,tcFile [,tcClass]])
Parameter
tlSingleClass
Mode of file pickup, allows processing of single classes.
Type Boolean
Direction Input

If true, pick up a single class via AGETCLASS() or Input on tcFile and tcClass.

If false, GetFile() any source file (Default), or tcFile.

FoxBin2Prg will run in appropriate mode.
tcFile
File to process.
Type Numeric
Direction In
Optional  

Any file except project, what must be processed with Convert_Pjx_2Bin.

Iput might be Binary or Text.

If tlSingleClass, in combination with tcClass.

Does not accept class in form library::class.

Do not add the structural index file, it will autoprocessed.
tcClass
Class name.
Type Numeric
Direction In
Optional  

Only with tcFile, if tlSingleClass is true.
Return value Boolean
Returns success of operation.
Remarks

Runs a pickup for a file and transforms it. Will always transform, neither the file is changed or not.

Will select the txt file fitting. Same for selecting a class, it will process the classes text file if existing.

See also
Convert_Pjx_2Bin
