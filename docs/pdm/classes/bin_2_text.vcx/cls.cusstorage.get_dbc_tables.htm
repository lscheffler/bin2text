get_dbc_tables Method
Class: cusstorage
Description:Return the tables of DBCs processed.
Resolve tables of databases.
RetVal 
=
Object.get_dbc_tables
(@taSource ,@taSearched ,@taReturn ,@tnOffset ,tcMode ,tlText2Bin ,toConverter)
Parameter
taSource
Source array.
Type Array
By reference  
Direction In
Array what is parsed for Databases. The databases found in column 1 where searched for tables.
taSearched
Array with tables.
Type Array
By reference  
Direction In
This array is checked for tables found in the databases. If an entry is found in column 1, the table will not be added to taReturn.
taReturn
Type Array
By reference  
Direction In/Out
Tables will be added to the array.

Table structure is the one of parameter taFiles of Convert

tnOffset
The last valid record in taReturn
Type Numeric
By reference  
Direction In/Out
tables will be appended after this row. The array will be resized to it's lenght. Will hold total row number at return.
tcMode
FoxBin2PRG's mode switch.
Type Character
Direction Input
"PRG2BIN","BIN2PRG" for the use of FoxBin2Prg.
tlText2Bin
Direction of operation,
Type Boolean
Direction In
To determin file extension.
toConverter
FoxBin2Prg object.
Type Object
Direction In
To get settings.
Return value Boolean
At least one table is added to taReturn
See also
Convert
