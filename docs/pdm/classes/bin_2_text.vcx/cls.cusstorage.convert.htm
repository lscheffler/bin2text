convert Method
Class: cusstorage
Description:Convert complete set, not partial.
Converts a preprocessed bunch of files with FoxBin2Prg.Execute
RetVal 
=
Object.convert
(taFiles ,tlProcess ,toConverter ,tnOffset [,toFB2T_Setting])
Parameter
taFiles
Array with files.
Type Array
Direction Input

A Three column array.

Column one
File to process.
Column two
Second parameter of FoxBin2PRG's Execute
Column three
Include file of colum 1 in output to screen
tlProcess
Value for FoxBin2PRG's l_ProcessFiles.
Type Boolean
Direction Input
toConverter
Converter object
Type Object
Direction Input
Instance of FoxBin2PRG for further operation.
tnOffset
Offset for progress bar.
Type Numeric
Direction Input
toFB2T_Setting
Settinngs object
Type Object
Direction Input
Optional  
A settings object as returned by FoxBin2PRgs Get_DirSettings. May be used to manipulate special settings.
Return value Boolean
Operation successfully ended.
