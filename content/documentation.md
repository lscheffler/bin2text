### Change log
See [Bin 2 Text Change Log](change_log.md) for a short change list.

---
### To use the program:
* Pull FoxBin2Prg
* Pull project
* Run `DO InitMenu IN LOCFILE('Bin2Text.app')` from VFP IDE
 * You will be asked for the location of _FoxBin2Prg.prg_
